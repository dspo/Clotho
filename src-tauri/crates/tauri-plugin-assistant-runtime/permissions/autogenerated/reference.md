## 默认权限

嵌入式 assistant runtime bridge 的默认权限。

#### 默认权限集包含：

- `allow-list-threads`
- `allow-get-thread-snapshot`
- `allow-create-thread`
- `allow-start-turn`
- `allow-resume-turn-stream`
- `allow-cancel-turn`
- `allow-submit-runtime-request`
- `allow-list-config-files`
- `allow-resolve-config-profile`
- `allow-get-runtime-catalog`
