```
│  ■ ■ ■  Clotho                                    ─  □  ×         │ ← 标题栏
├────────┬────────────────────────────────────────────────────────────┤

│  边    │  └───────────────────────────────────┘  └─────────────┘  │
│  栏    │  ┌─ 视图切换 Tab ────────────────────────────────────┐   │
│        │  │ [看板] [列表] [甘特图] [日历]                       │   │
│        │  └──────────────────────────────────────────────────┘   │

│        │  │                                                    │   │

```

## 侧边栏

- 默认宽度：240px
- 可拖拽调整：最小 200px，最大 400px
- 支持折叠为图标模式（48px）
- 折叠快捷键：Cmd+\

### 侧边栏内容

1. 搜索框（Cmd+K 命令面板）
2. 收藏夹
3. 项目列表
4. 标签列表
5. 设置

## 主题系统

### 亮色主题

- 侧边栏：#F8F9FA
- 主内容区：#FFFFFF
- 文字：#1A1A1A
- 强调色：#3B82F6

### 暗色主题

- 侧边栏：#1E1E2E
- 主内容区：#24243A
- 文字：#E0E0E0
- 强调色：#60A5FA

详细设计请参阅 [overview.md](./overview.md) 第 5 节。
