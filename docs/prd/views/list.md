# 列表视图 - 详细设计文档

## 1. 概述

列表视图是 Clotho 的核心视图之一，采用表格式布局展示任务数据。用户可在表格中直接编辑任务属性、排序、筛选、分组，适合需要快速浏览和批量操作大量任务的场景。

技术实现基于 TanStack Table（headless 表格逻辑）+ TanStack Virtual（虚拟滚动），渲染层使用 shadcn/ui 的表格样式。

---

## 2. 表格布局

### 2.1 默认列定义

| 列序号 | 列标识 | 列名 | 默认宽度 | 可调整 | 可排序 | 可编辑 | 固定 |
|-------|--------|------|---------|--------|--------|--------|------|
| 1 | checkbox | （复选框） | 32px | 否 | 否 | 否 | 左侧 |
| 2 | title | 任务名称 | flex（撑满剩余空间） | 是 | 是 | 是 | 左侧 |
| 3 | status | 状态 | 120px | 是 | 是 | 是 | 否 |
| 4 | priority | 优先级 | 100px | 是 | 是 | 是 | 否 |
| 5 | dueDate | 截止日期 | 120px | 是 | 是 | 是 | 否 |
| 6 | tags | 标签 | 150px | 是 | 否 | 是 | 否 |
| 7 | project | 项目 | 120px | 是 | 是 | 否 | 否 |

**列固定行为**：复选框列和任务名称列固定在左侧，水平滚动时始终可见。固定列与滚动区域之间用 1px 投影分隔线区分。

### 2.2 列选择器

- 入口：工具栏中的眼睛图标按钮（`Eye` Lucide 图标）
- 点击弹出 Popover 面板，内含每列对应的 checkbox
- 复选框列和任务名称列不可隐藏（checkbox 禁用且勾选）
- 勾选/取消即时生效，无需确认
- 列顺序不可通过选择器调整（固定顺序）

```
┌─────────────────┐
│  显示列           │
├─────────────────┤
│  ☑ 任务名称（锁定）│
│  ☑ 状态           │
│  ☑ 优先级         │
│  ☑ 截止日期       │
│  ☐ 标签           │
│  ☑ 项目           │
└─────────────────┘
```

### 2.3 列宽调整

- 列头右边框区域（4px 热区）可拖拽调整列宽
- 鼠标悬停时光标变为 `col-resize`
- 拖拽过程中显示蓝色竖线指示当前宽度
- 最小列宽 60px，无最大限制
- 任务名称列（flex 列）不可手动调宽，自动填充剩余空间

### 2.4 列头排序

- 点击列头文字区域触发排序
- 三态循环：无排序 → 升序（↑） → 降序（↓） → 无排序
- 排序图标显示在列名右侧，无排序时隐藏
- 同时只支持单列排序（点击新列重置旧列排序）
- 排序状态字段优先级顺序：Backlog < Todo < In Progress < Done < Cancelled
- 排序优先级字段顺序：Urgent > High > Medium > Low

## 3. 行设计

### 3.1 行基础样式

| 属性 | 值 |
|------|-----|
| 行高 | 40px |
| 行间距 | 0（紧凑排列） |
| 行间分隔 | 底部 1px border（border-border） |
| 文字大小 | 14px（text-sm） |
| 内边距 | 水平 8px |

### 3.2 行状态样式

| 状态 | 背景色 | 附加样式 |
|------|--------|---------|
| 默认 | transparent | — |
| 悬停（hover） | muted/50 | — |
| 选中（selected） | accent/10 | 左侧 2px accent 色条 |
| 选中+悬停 | accent/15 | 左侧 2px accent 色条 |
| 编辑中 | background | 编辑单元格高亮 border |

### 3.3 子任务缩进

- 每级子任务左侧缩进 24px（仅在任务名称列体现）
- 有子任务的行在名称前显示展开/折叠箭头（`ChevronRight` / `ChevronDown`）
- 箭头点击区域 24x24px
- 无子任务的行不显示箭头，名称与有子任务的行对齐（箭头位置留空）
- 折叠状态下子任务行隐藏，父任务名称后显示子任务数量角标（灰色小字 `(3)`）
- 默认展开第一层子任务

```
任务名称列内部布局（有子任务的父任务）：
┌─[缩进 0px]─[▼ 24px]─[任务标题文字]─────────────────────┐

任务名称列内部布局（子任务）：
┌─[缩进 24px]──[24px 留空]─[子任务标题文字]──────────────────┐

任务名称列内部布局（有子任务的子任务）：
┌─[缩进 24px]──[▶ 24px]─[子任务标题文字]─(2)─────────────────┐
```

## 4. 单元格内容渲染

### 4.1 复选框列

- 居中显示 shadcn/ui `Checkbox` 组件
- 大小 16x16px
- 列头处的复选框为全选/全不选控制（三态：未选、部分选中、全选）

### 4.2 任务名称列

- 显示任务标题文本，单行，超出省略号截断
- 文字颜色：foreground（默认）
- 已完成任务（Done）：文字添加删除线 + 颜色变为 muted-foreground
- 已取消任务（Cancelled）：文字添加删除线 + 颜色变为 muted-foreground

### 4.3 状态列

- 显示为状态圆点（8px）+ 状态文字
- 圆点颜色对应状态颜色（同 PRD 定义）：
  - Backlog：#6B7280（灰）
  - Todo：#8B5CF6（紫）
  - In Progress：#F59E0B（琥珀）
  - Done：#10B981（绿）
  - Cancelled：#EF4444（红）

### 4.4 优先级列

- 显示为优先级图标 + 文字
- 图标：使用不同填充的旗帜图标（`Flag` Lucide），颜色对应优先级：
  - Urgent：#EF4444（红）
  - High：#F97316（橙）
  - Medium：#EAB308（黄）
  - Low：#3B82F6（蓝）

### 4.5 截止日期列

- 格式：`MM/DD`（同年）或 `YYYY/MM/DD`（跨年）
- 已过期且未完成：文字变为红色（destructive）
- 今天到期：文字变为琥珀色（#F59E0B）
- 无截止日期：显示 `—`

### 4.6 标签列

- 显示为标签 Badge 列表（shadcn/ui `Badge` variant=outline）
- 每个 Badge 左侧有标签颜色圆点（6px）
- 单元格内最多显示 2 个标签，超出显示 `+N`
- 无标签：留空

### 4.7 项目列

- 显示项目颜色圆点（8px）+ 项目名称
- 文字单行截断

## 5. 行内编辑

### 5.1 触发方式

- 双击单元格进入编辑模式
- 快捷键 Enter 对当前选中行的任务名称进入编辑

### 5.2 各字段编辑组件

| 字段 | 编辑组件 | 交互说明 |
|------|---------|---------|
| 任务名称 | 文本输入框（Input） | 双击后文字变为 Input，自动全选文字，Enter 保存，Esc 取消 |
| 状态 | 下拉选择（Select） | 双击弹出 5 种状态下拉列表，选中即保存，每项前有状态颜色圆点 |
| 优先级 | 下拉选择（Select） | 双击弹出 4 级优先级下拉列表，选中即保存，每项前有优先级颜色图标 |
| 截止日期 | 日期选择器（DatePicker Popover） | 双击弹出日历面板（shadcn/ui DatePicker），选择日期即保存 |
| 标签 | 标签选择器（Popover + Combobox） | 双击弹出多选标签面板，支持搜索、创建新标签，点击外部保存 |

### 5.3 键盘导航

| 快捷键 | 行为 |
|--------|------|
| Tab | 跳到同行下一个可编辑单元格 |
| Shift+Tab | 跳到同行上一个可编辑单元格 |
| Enter | 保存当前编辑并退出编辑模式 |
| Escape | 取消编辑，恢复原值 |

Tab 顺序：任务名称 → 状态 → 优先级 → 截止日期 → 标签

### 5.4 编辑态视觉

- 编辑中的单元格显示 2px accent 色 border
- 编辑中的单元格背景变为 background（纯白/纯黑）
- 同行其余单元格保持原样

## 6. 多选与批量操作

### 6.1 选择方式

| 操作 | 行为 |
|------|------|
| 点击复选框 | 切换单行选中状态 |
| Shift+Click 复选框 | 范围选择：从上一次点击的行到当前行之间所有行 |
| Cmd+A | 全选当前分组内所有任务（如无分组则全选所有可见任务） |
| 点击列头复选框 | 全选/全不选 |
| Escape | 取消所有选择 |

### 6.2 批量操作栏

选中 1 项或以上时，表格顶部（列头上方）浮出批量操作栏，覆盖在工具栏之上。

```
┌─────────────────────────────────────────────────────────────────┐
│  ☑ 已选 3 项  │ 修改状态 ▾ │ 修改优先级 ▾ │ 添加标签 │ 删除 │ ✕ │
└─────────────────────────────────────────────────────────────────┘
```

| 操作 | 组件 | 行为 |
|------|------|------|
| 已选 N 项 | 文字 | 显示当前选中数量 |
| 修改状态 | 下拉菜单 | 批量设置选中任务的状态 |
| 修改优先级 | 下拉菜单 | 批量设置选中任务的优先级 |
| 添加标签 | 弹窗 | 为选中任务添加标签（追加，非覆盖） |
| 删除 | 按钮（destructive） | 批量删除，需二次确认对话框 |
| ✕ | 关闭按钮 | 取消所有选择，隐藏批量操作栏 |

### 6.3 批量操作反馈

- 操作完成后显示 Toast：「已更新 N 项任务」
- 删除操作 Toast 提供「撤销」按钮（5 秒有效）

## 7. 分组功能

### 7.1 分组选项

入口：工具栏「分组」下拉按钮（`Group` Lucide 图标）。

| 选项 | 分组依据 | 分组列表 |
|------|---------|---------|
| 无分组 | — | 所有任务平铺 |
| 按状态 | task.status | Backlog / Todo / In Progress / Done / Cancelled |
| 按优先级 | task.priority | Urgent / High / Medium / Low |
| 按项目 | task.project_id | 各项目名称 |
| 按标签 | task.tags | 各标签名 + 「无标签」组 |

### 7.2 分组头设计

```
┌──────────────────────────────────────────────────────────────────┐
│  ▼  ● In Progress   3 项任务                                     │
├──────────────────────────────────────────────────────────────────┤
│  （该分组下的任务行）                                               │
└──────────────────────────────────────────────────────────────────┘
```

| 元素 | 说明 |
|------|------|
| 折叠/展开箭头 | `ChevronDown`（展开）/ `ChevronRight`（折叠），点击切换 |
| 分组标识 | 颜色圆点（状态/优先级颜色）或项目图标 |
| 分组名称 | 状态名/优先级名/项目名/标签名，14px semibold |
| 任务计数 | 灰色小字，如 `3 项任务` |

- 分组头高度：36px
- 分组头背景色：muted/30
- 分组头不可编辑
- 分组默认全部展开
- 分组头可拖拽调整顺序（拖拽时整组移动）
- 分组内独立排序（列头排序仅作用于分组内）

### 7.3 空分组

- 分组值存在但无任务时，分组头仍显示，计数为 `0 项任务`
- 可通过分组底部的「+ 添加任务」在该分组内创建任务

## 8. 筛选与搜索

### 8.1 搜索框

- 位置：工具栏右侧，Search 图标 + 输入框
- 快捷键：Cmd+F 聚焦搜索框
- 搜索范围：任务标题（模糊匹配）
- 防抖：300ms
- 搜索结果即时过滤表格行，匹配文字高亮（黄色背景）
- 清空搜索框恢复全部显示
- 搜索框右侧显示匹配数量（如 `3 / 28`）

### 8.2 筛选面板

- 入口：工具栏「筛选」按钮（`Filter` Lucide 图标）
- 点击弹出 Popover 面板

```
┌───────────────────────────────────────────────┐
│  筛选条件                                       │
├───────────────────────────────────────────────┤
│  ┌─────────┐ ┌────────┐ ┌──────────────┐     │
│  │ 状态   ▾ │ │ 等于 ▾ │ │ In Progress ▾│  ✕  │
│  └─────────┘ └────────┘ └──────────────┘     │
│  ┌─────────┐ ┌────────┐ ┌──────────────┐     │
│  │ 优先级 ▾ │ │ 等于 ▾ │ │ High       ▾│  ✕  │
│  └─────────┘ └────────┘ └──────────────┘     │
│                                               │
│  + 添加筛选条件                                 │
└───────────────────────────────────────────────┘
```

**筛选条件结构**：字段 + 操作符 + 值

| 字段 | 可用操作符 | 值类型 |
|------|----------|--------|
| 状态 | 等于 / 不等于 | 单选下拉（5 种状态） |
| 优先级 | 等于 / 不等于 | 单选下拉（4 种优先级） |
| 截止日期 | 早于 / 晚于 / 等于 / 在...之间 | 日期选择器 |
| 标签 | 包含 / 不包含 | 多选标签 |
| 项目 | 等于 / 不等于 | 单选下拉（项目列表） |

- 多个条件之间为 AND 逻辑
- 每个条件行右侧有删除按钮（✕）
- 点击「+ 添加筛选条件」追加新行

### 8.3 活跃筛选展示

- 活跃筛选条件以 Chip 形式显示在工具栏下方
- 每个 Chip 显示：`字段: 值`（如 `状态: In Progress`）
- Chip 右侧有 ✕ 可单独删除该条件
- 所有 Chip 最右侧有「清除全部」按钮

```
工具栏下方：
┌────────────────┐ ┌──────────────┐
│ 状态: In Progress ✕ │ │ 优先级: High ✕ │  清除全部
└────────────────┘ └──────────────┘
```

## 9. 虚拟滚动

### 9.1 TanStack Virtual 配置

| 参数 | 值 | 说明 |
|------|-----|------|
| estimateSize | 40px | 行高估算值 |
| overscan | 5 | 可视区域上下各多渲染 5 行缓冲 |
| scrollPaddingStart | 0 | 表头高度由外部处理 |

### 9.2 性能目标

- 滚动帧率：60fps
- 支持 10000+ 任务行无卡顿
- 首屏渲染时间 < 100ms

### 9.3 实现要点

- 表格容器设置固定高度（填满主内容区剩余空间）
- 表头独立于虚拟滚动区域（始终可见）
- 分组头参与虚拟滚动（作为特殊行类型）
- 滚动条使用原生滚动条（与系统风格一致）

## 10. 行末快速操作

### 10.1 悬停操作图标

鼠标悬停任务行时，行末显示操作图标（覆盖在最后一列上方）：

```
                                              ┌─────┐
│ ... │ 标签列内容 │ 12/25   │  │ ↗  ··· │  ← 悬停时出现

```

| 图标 | 功能 | 说明 |
|------|------|------|
| ↗ (`ExternalLink`) | 打开详情 | 点击打开右侧任务详情面板 |
| ··· (`MoreHorizontal`) | 更多操作 | 点击弹出下拉菜单 |

### 10.2 更多操作菜单

```
┌──────────────────┐
│  打开详情          │
│  编辑标题          │
│  ─────────────── │
│  设置状态     ▸   │

│  删除任务          │
└──────────────────┘
```

菜单同时支持右键触发（在任务行任意位置右键）。

## 11. 新建任务

### 11.1 底部新建行

列表底部始终显示一行新建行，样式与普通行区分：

```
├───┼────────────────────┼────────┼────────┼────────┼──────────┤
│   │  + 添加任务...       │        │        │        │          │
└───┴────────────────────┴────────┴────────┴────────┴──────────┘
```

- 文字 `+ 添加任务...` 使用 muted-foreground 颜色
- 点击后任务名称列变为 Input 输入框，自动聚焦
- 输入标题后按 Enter 创建任务：
  - 如当前有分组，继承当前分组的属性（如在 `In Progress` 分组下创建，状态自动设为 In Progress）
  - 如无分组，默认状态为 `Todo`，优先级为 `Low`
  - 创建成功后 Input 清空，焦点保持，可连续创建
- Escape 取消输入，退出编辑状态

### 11.2 快捷键创建

- 按 `C` 键（非编辑状态下）聚焦底部新建行的 Input

## 15. 边界场景处理

### 15.1 空状态

当项目内无任何任务时：

```
┌──────────────────────────────────────────────────────┐
│                                                      │
│              📋                                       │
│              还没有任务                                │
│              创建你的第一个任务开始吧                    │
│                                                      │
│              [+ 创建任务]                              │
│                                                      │
└──────────────────────────────────────────────────────┘
```

### 15.2 筛选无结果

```
┌──────────────────────────────────────────────────────┐
│                                                      │
│              🔍                                       │
│              没有匹配的任务                             │
│              尝试调整筛选条件                           │
│                                                      │
│              [清除筛选]                                │
│                                                      │
└──────────────────────────────────────────────────────┘
```

### 15.3 加载状态

- 首次加载：显示 3-5 行骨架屏（Skeleton），模拟表格行
- 排序/筛选切换：表格内容区 opacity 降至 0.5 + 顶部细线 loading bar

### 15.4 操作冲突

- 编辑中点击其他行：自动保存当前编辑，切换到新行
- 编辑中触发排序/筛选：自动保存当前编辑，应用排序/筛选
- 批量选中时双击编辑：退出批量选择模式，进入编辑模式
