# Task Detail 窗口

## 布局
- 类型：浮动窗口 (Modal Dialog)
- 非系统级窗口，前端实现
- 居中显示，带半透明遮罩层

## 尺寸
- 默认宽度：896px (max-w-4xl)
- 默认高度：视口 80%
- 最小宽度：640px
- 最小高度：480px

## 内容布局
┌──────────────────────────────────────────────┐
│  [X]                              任务标题     │
├──────────────────────────────────────────────┤
│  状态: [Todo ▾]   优先级: [Medium ▾]           │
│  开始: [日期]     截止: [日期]                  │
├──────────────────────────────────────────────┤
│                                               │
│  Description                                  │
│  ┌─────────────────────────────────────────┐ │
│  │  (富文本/Markdown 编辑器)                  │ │
│  │  最小高度 400px                           │ │
│  └─────────────────────────────────────────┘ │
│                                               │
├──────────────────────────────────────────────┤
│  子任务 (3/5)                                 │
│  ☑ 子任务1  ☐ 子任务2  ☐ 子任务3              │
├──────────────────────────────────────────────┤
│  标签: [Bug] [Frontend] [+添加]               │
└──────────────────────────────────────────────┘

## 交互
- ESC 或点击遮罩层关闭
- 修改自动保存
- 关闭时无需确认（已自动保存）
